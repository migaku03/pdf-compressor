PDF 圧縮ツール

Ghostscriptを使ったローカル動作のPDF圧縮ツールです。ファイルは外部に送信されません。

Ghostscriptが見つかりません
このツールの動作にはGhostscriptのインストールが必要です。
Windows: winget install Ghostscript または 公式サイトからダウンロード
Mac: brew install ghostscript
Linux: sudo apt install ghostscript
