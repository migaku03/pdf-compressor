# Claude Code 行動指針

## 基本方針
- 問題が発生したら自分で原因を分析・修正・確認まで完結させる
- ユーザーには最終結果のみ報告する
- 不明点があっても自分でベストな判断をして進める

## 実装・修正時の必須手順
1. 実装・修正を行う
2. python3 -m py_compile app.py で構文チェック
3. 想定されるエラーを自分でリストアップして潰す
4. ./run.sh で起動確認
5. 問題があれば自己修正してから報告

## 報告フォーマット
- 何を修正したか（1〜2行）
- 動作確認結果（OK/NG）
- ユーザーがやること（最終確認のみ）

## 禁止事項
- 未確認のまま「試してください」と言わない
- エラーの原因を「〜の可能性があります」で終わらせない
- ユーザーにデバッグ作業をさせない
